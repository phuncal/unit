# Unit

macOS 桌面聊天工具，专为与 AI 深度讨论游戏、影视、文学作品的设定内容而设计。

## 特性

- 兼容 OpenAI 格式的流式接口
- 设定档案系统（archive.md）
- 滑动窗口上下文，优化长对话成本
- 发送前费用预估与累计统计
- 对话模板与消息锚点
- 图片支持与双版本导出

## 技术栈

Electron + React + TypeScript + Tailwind CSS + IndexedDB (Dexie.js)

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build:mac
```
