Core Features

Designed for
long conversations.

Every feature exists because deep creative work demands it. Not because it could be added.

01

Living Archive

Conclusions persist across sessions in a local archive.md file. Each new conversation in the same project picks up exactly where thinking left off.

02

Anchor Messages

Pin critical context — a design document, a core premise — and it travels with every request, immune to the sliding window.

03

Token Awareness

See cost estimates before sending. Track accumulated spend per conversation. Sliding context windows keep long discussions economical.

04

Any API, Any Model

Connect to Anthropic, OpenRouter, or any OpenAI-compatible endpoint. Fetch available models automatically. Keys stay local, encrypted in system keychain.

05

Export to Markdown

Long outputs save directly as .md files. Literary documents convert to structured technical specs. Versions merge cleanly into new files.

06

Conversation Templates

Save system prompts as reusable roles — game design critic, worldbuilding collaborator, narrative analyst. Start deep immediately.

Workflow

A better way to
build worlds.

Unit removes the friction between thinking and documenting. Your project folder becomes the single source of truth.

1

Bind a folder

Point Unit to your project directory. An archive.md is created automatically to hold everything that matters.

2

Upload & discuss

Drag in reference documents. Anchor the critical ones. Talk through your ideas with full context intact.

3

Update archive

One click extracts confirmed conclusions into the archive. You review each entry. Nothing gets in without your approval.

4

Export & build

Convert creative documents to technical specs. Export directly to your project folder. Claude Code reads them immediately.

Design Philosophy

"Good design is
as little design
as possible."

— Dieter Rams

01

Nothing extraneous

No cloud sync. No accounts. No telemetry. No web search. No plugin system. Unit does one thing and protects the focus required to do it well.

02

Strict isolation

Each conversation exists independently. No cross-contamination between projects. The AI sees only what you've chosen to show it.

03

Your data, your machine

Everything lives in folders you own. API keys are encrypted in the system keychain. The only data leaving your computer is the message you chose to send.

04

Built for endurance

Virtual scrolling handles thousands of messages. Streaming output never blocks. Large file exports never freeze. Long work deserves a reliable tool.
